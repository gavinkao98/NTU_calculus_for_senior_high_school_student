\chapter{Inverse Functions and Limits}

This chapter develops two themes that together form the starting point of calculus. The first is the idea of reversing a function. When a process sends each input to a distinct output, we can ask how to recover the input from the output. This leads to the notions of one-to-one functions and inverse functions, and in particular to the inverse trigonometric functions, which appear throughout geometry, physics, and later differentiation formulas.

The second theme is the limit of a function. Many quantities of interest in calculus---the instantaneous rate of change of a moving object, the slope of a curve at a point, the behavior of a function near a hole in its graph---cannot be read off from a single value of the function. They are determined by the behavior of the function \emph{near} a point, not \emph{at} the point. We will first study limits informally by looking at tables and graphs, then extend the idea to one-sided limits, infinite limits, and their geometric counterpart, vertical asymptotes. After developing algebraic techniques for computing limits, we close the chapter with the precise \(\varepsilon\)-\(\delta\) definition, which places all of the preceding discussion on a rigorous foundation and prepares the way for derivatives and integrals in the chapters that follow.

\paragraph{By the end of this chapter, you will be able to:}
\begin{itemize}
    \item decide whether a given function is one-to-one, and construct its inverse when it exists;
    \item work with the inverse trigonometric functions \(\arcsin\), \(\arccos\), \(\arctan\), and their companions, together with their principal ranges;
    \item estimate the limit of a function from tables, graphs, and algebraic simplification, including one-sided and infinite limits;
    \item compute limits using the limit laws, the squeeze theorem, and standard techniques such as factoring and rationalization;
    \item state and apply the precise \(\varepsilon\)-\(\delta\) definition of a limit, and use it to verify a given limit rigorously.
\end{itemize}

\section{Inverse Functions}

Not every function can be reversed. If two different inputs produce the same output, then the original input cannot be uniquely recovered from that output. For this reason, the existence of an inverse function depends on a special property called one-to-one. In this section, we study one-to-one functions and then introduce inverse functions.

\subsection{One-to-one functions}

A function can have an inverse only when each output comes from exactly one input. This motivates the following definition.

\begin{definition}
Let \(f\) be a function with domain \(A\). We say that \(f\) is \emph{one-to-one}\index{one-to-one function} if
\[
f(x_1)\ne f(x_2)
\qquad \text{whenever } x_1\ne x_2.
\]
Equivalently, \(f\) is one-to-one if
\[
f(x_1)=f(x_2) \implies x_1=x_2.
\]
\emph{Informally, a function is one-to-one when different inputs always produce different outputs.}
\end{definition}

\begin{workedexample}
\begin{example}
Consider the following functions defined on a group of students:
\begin{aligneddisplay}
f(\text{student}) &= \text{Student ID number},\\
g(\text{student}) &= \text{Blood type}.
\end{aligneddisplay}
Determine which of these real-world functions are one-to-one.
\end{example}

\begin{solution}
The function \(f\) is one-to-one because each student has a unique ID number, so different students always produce different outputs.

The function \(g\) is not one-to-one because different students can share the same blood type. For example, if Alice and Bob are both Type O, then
\pairdisplay{g(\text{Alice})=\text{Type O},}{g(\text{Bob})=\text{Type O}.}
Since two different inputs give the same output, \(g\) is not one-to-one.
\end{solution}
\end{workedexample}

Now let us consider some mathematical examples.

\begin{workedexample}
\begin{example}
Consider the functions
\begin{conditiondisplay}
f(x) &= x, \qquad & 0\le x\le 1,\\
g(x) &= x^2, \qquad & -1\le x\le 1.
\end{conditiondisplay}
Determine which of these functions are one-to-one.
\end{example}

\begin{solution}
The function \(f(x)=x\) is one-to-one on \([0,1]\) because different inputs always produce different outputs.

The function \(g(x)=x^2\) is not one-to-one on \([-1,1]\) because
\begin{aligneddisplay}
g\left(\frac12\right) &= \left(\frac12\right)^2=\frac14,\\
g\left(-\frac12\right) &= \left(-\frac12\right)^2=\frac14.
\end{aligneddisplay}
Thus two different inputs give the same output, so \(g\) is not one-to-one.
\end{solution}
\end{workedexample}

Not every function has an inverse. A function can have an inverse only if each output corresponds to exactly one input; in other words, the function must be one-to-one. For \(g(x)=x^2\) on \([-1,1]\), the value \(\frac14\) comes from both \(\frac12\) and \(-\frac12\). Therefore \(g^{-1}\!\left(\frac14\right)\) is not well defined.

\begin{remark}[Horizontal Line Test]\index{horizontal line test}
A function is one-to-one if and only if no horizontal line intersects its graph more than once. A horizontal line has equation \(y=c\), so it represents a fixed output value \(c\). If the line \(y=c\) intersects the graph of \(f\) at two distinct points \((x_1,c)\) and \((x_2,c)\) with \(x_1\ne x_2\), then
\[
f(x_1)=c=f(x_2).
\]
Thus two different inputs produce the same output, so \(f\) is not one-to-one. Conversely, if every horizontal line intersects the graph at most once, then no output value corresponds to more than one input, and hence \(f\) is one-to-one. This criterion is illustrated in \cref{fig:horizontal-line-test}.
\end{remark}

\subsection{Inverse functions}

Suppose that \(f\) is a one-to-one function. If \(f\) maps \(x\) to \(y\), then its inverse maps \(y\) back to \(x\).

\begin{definition}
Let \(f\) be a one-to-one function with domain \(A\) and range \(B\). Then the \emph{inverse function}\index{inverse function}\index{inverse function@$f^{-1}$} of \(f\), denoted by \(f^{-1}\), has domain \(B\) and range \(A\), and is defined by
\[
f^{-1}(y)=x \quad \Longleftrightarrow \quad f(x)=y \ \text{and}\ y\in B.
\]
\end{definition}

When we focus on the inverse function, it is common to use \(x\) as the independent variable. In that case we may write
\[
f^{-1}(x)=y
\quad \Longleftrightarrow \quad
f(y)=x.
\]

\begin{theorem}
A function has an inverse function if and only if it is one-to-one.
\end{theorem}

\begin{proof}
Suppose first that \(f\) has an inverse function \(f^{-1}\). If \(f(x_1)=f(x_2)\), then applying \(f^{-1}\) to both sides gives
\[
f^{-1}(f(x_1))=f^{-1}(f(x_2)),
\]
so \(x_1=x_2\). Hence \(f\) is one-to-one.

Conversely, suppose that \(f\) is one-to-one with domain \(A\) and range \(B\). For each \(y\in B\), there is some \(x\in A\) such that \(f(x)=y\), and the one-to-one property guarantees that this \(x\) is unique. Therefore the correspondence \(y\mapsto x\) defines an inverse function \(f^{-1}:B\to A\).
\end{proof}

\begin{workedexample}
\begin{example}
Find the inverse of each function.
\begin{enumerate}
    \item \(f(x)=x\)
    \item \(g(x)=x^3\)
\end{enumerate}
\end{example}

\begin{solution}
\solutionbreak
\begin{enumerate}
    \item Since \(f(x)=x\) leaves every input unchanged, it is its own inverse, so \(f^{-1}(x)=x\).

    \item To find the inverse of \(g\), let \(y=x^3\). Solving for \(x\) gives \(x=\sqrt[3]{y}\). Interchanging \(x\) and \(y\) then gives \(g^{-1}(x)=\sqrt[3]{x}\).
\end{enumerate}
\end{solution}
\end{workedexample}

\begin{proposition}
Let \(f\) be a one-to-one function with domain \(A\) and range \(B\). Then
\pairdisplay{f^{-1}(f(x))=x,\quad \text{for every } x\in A,}{f(f^{-1}(x))=x,\quad \text{for every } x\in B.}
\end{proposition}

\Cref{fig:inverse-composition} illustrates these two composition identities.

\subsection{Finding the inverse of a function}

When \(f\) is one-to-one, the inverse can almost always be constructed by a short three-step procedure.

\begin{strategy}[Finding the inverse of a one-to-one function]
\begin{enumerate}
    \item Write \(y=f(x)\).
    \item Solve this equation for \(x\) in terms of \(y\).
    \item Interchange \(x\) and \(y\). The resulting equation gives \(y=f^{-1}(x)\).
\end{enumerate}
\end{strategy}

\begin{workedexample}
\begin{example}
Find the inverse of \(f(x)=x^3+2\).
\end{example}

\begin{solution}
Let \(y=x^3+2\). Solving for \(x\) gives
\begin{aligneddisplay}
x^3 &= y-2,\\
x &= \sqrt[3]{\,y-2\,}.
\end{aligneddisplay}
Interchanging \(x\) and \(y\) gives \(f^{-1}(x)=\sqrt[3]{\,x-2\,}\).
To verify this, compute
\begin{aligneddisplay}
f\bigl(f^{-1}(x)\bigr)
&=
\left(\sqrt[3]{\,x-2\,}\right)^3+2\\
&=
x-2+2\\
&=
x.
\end{aligneddisplay}
Thus the inverse is correct.
\end{solution}
\end{workedexample}

\begin{workedexample}
\begin{example}
Find the inverse of \(f(x)=x^2\) for \(x\ge 0\).
\end{example}

\begin{solution}
On the domain \([0,\infty)\), the function \(f(x)=x^2\) is one-to-one, so an inverse exists. Let \(y=x^2\) with \(x\ge 0\).
Solving for \(x\) gives \(x=\sqrt{y}\). Because the original domain requires \(x\ge 0\), we take the nonnegative square root. Interchanging \(x\) and \(y\) then gives
\begin{conditiondisplay}
f^{-1}(x) &= \sqrt{x}, \qquad & x\ge 0.
\end{conditiondisplay}
This example shows that a function that is not one-to-one on all of \(\mathbb{R}\) may still have an inverse after its domain is restricted appropriately.
\end{solution}
\end{workedexample}

\section{Inverse Trigonometric Functions}

The trigonometric functions are not one-to-one on all of \(\mathbb{R}\), so they do not have inverse functions on their full domains. Inverse trigonometric functions arise whenever we know a ratio and want to recover an angle. This occurs in elementary geometry, surveying, navigation, and physics. For example, if we know the slope of a ramp, then \(\arctan\) can be used to determine its angle of inclination.

To define an inverse trigonometric function, we must restrict the domain of the original trigonometric function to an interval on which it is one-to-one. The chosen interval determines the principal values of the inverse function.

Consider the sine function on all of \(\mathbb{R}\). As shown in \cref{fig:sine-not-one-to-one}, a horizontal line can intersect the graph of \(y=\sin x\) more than once. Therefore \(\sin x\) is not one-to-one on \(\mathbb{R}\), and so it does not have an inverse function on its full domain.

\subsection{The inverse sine function}

To define the inverse sine function, we restrict \(\sin x\) to the interval
\[
-\frac{\pi}{2}\le x\le \frac{\pi}{2}.
\]
On this interval, \(\sin x\) is one-to-one, and every value in the range \([-1,1]\) is attained. The restricted branch is shown in \cref{fig:restricted-sine}.

\begin{definition}
The \emph{inverse sine function}\index{inverse sine function}\index{arcsine@$\arcsin$}, denoted by \(\arcsin x\), is defined by
\[
\arcsin x=y \quad \Longleftrightarrow \quad \sin y=x \ \text{and}\ -\tfrac{\pi}{2}\le y\le \tfrac{\pi}{2}.
\]
\end{definition}

The domain of \(\arcsin x\) is \([-1,1]\), and its range is \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\).

\begin{caution}\index{sine inverse vs reciprocal@$\sin^{-1}$ vs $1/\sin$}
The notation \(\sin^{-1}x\) is also used for \(\arcsin x\). It does \emph{not} mean the reciprocal \((\sin x)^{-1}=1/\sin x\). Whenever a source uses \(\sin^{-1}\), check from context which is meant.
\end{caution}

The composition identities for \(\arcsin\) take the following form.

\begin{proposition}[Inverse sine identities]
\begin{conditiondisplay}
\sin(\arcsin x) &= x, \qquad & -1\le x\le 1,\\
\arcsin(\sin x) &= x, \qquad & -\frac{\pi}{2}\le x\le \frac{\pi}{2}.
\end{conditiondisplay}
\end{proposition}

\begin{caution}\index{arcsin of sin@$\arcsin(\sin x)$ outside principal interval}
The identity \(\arcsin(\sin x)=x\) holds only when \(x\) lies in the principal interval \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\). For example,
\[
\arcsin(\sin \pi)=\arcsin(0)=0\ne \pi,
\]
because \(\arcsin x\) always returns a value in \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\).
\end{caution}

\begin{workedexample}
\begin{example}
Evaluate the following expressions.
\begin{enumerate}
    \item \(\arcsin\left(\frac{1}{2}\right)\)
    \item \(\tan\left(\arcsin\left(\frac{1}{3}\right)\right)\)
\end{enumerate}
\end{example}

\begin{solution}
\solutionbreak
\begin{enumerate}
    \item We need the angle \(y\) in the interval \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\) such that \(\sin y=\frac{1}{2}\). Since \(\sin\left(\frac{\pi}{6}\right)=\frac{1}{2}\), it follows that \(\arcsin\left(\frac{1}{2}\right)=\frac{\pi}{6}\).

    \item Let \(\theta=\arcsin\left(\frac{1}{3}\right)\). Then
    \begin{conditiondisplay}
    \sin\theta &= \frac{1}{3}, \qquad & -\frac{\pi}{2}\le \theta\le \frac{\pi}{2}.
    \end{conditiondisplay}
    Since \(\sin\theta>0\), the angle \(\theta\) lies in the first quadrant. A right triangle with hypotenuse \(3\) and opposite side \(1\) therefore has adjacent side \(a\).

    \begin{center}
    \begin{tikzpicture}[scale=1.1]
    \coordinate (A) at (0,0);
    \coordinate (B) at (3,0);
    \coordinate (C) at (3,1);
    \draw[thick] (A) -- (B) -- (C) -- cycle;
    \draw (2.75,0) -- (2.75,0.25) -- (3,0.25);
    \node[left] at (1.2,0.18) {\(\theta\)};
    \node[below] at (1.5,0) {\(a\)};
    \node[right] at (3,0.5) {\(1\)};
    \node[above left] at (1.45,0.5) {\(3\)};
    \end{tikzpicture}
    \end{center}

    By the Pythagorean Theorem,
    \[
    a=\sqrt{3^{2}-1^{2}}=\sqrt{8}=2\sqrt{2}.
    \]
    Therefore, \(\tan\left(\arcsin\left(\frac{1}{3}\right)\right)=\tan\theta=\frac{1}{a}=\frac{1}{2\sqrt{2}}\).
\end{enumerate}
\end{solution}
\end{workedexample}

\[
\arccos x=y \quad \Longleftrightarrow \quad \cos y=x \ \text{and}\ 0\le y\le \pi.

\begin{proposition}[Inverse cosine identities]
\begin{conditiondisplay}
\cos(\arccos x) &= x, \qquad & -1\le x\le 1,\\
\arccos(\cos x) &= x, \qquad & 0\le x\le \pi.
\end{conditiondisplay}
\end{proposition}

\begin{definition}
The \emph{inverse tangent function}\index{inverse tangent function}\index{arctangent@$\arctan$}, denoted by \(\arctan x\), is defined by
\[
\arctan x=y \quad \Longleftrightarrow \quad \tan y=x \ \text{and}\ -\tfrac{\pi}{2}<y<\tfrac{\pi}{2}.
\]
\end{definition}

The domain of \(\arctan x\) is \(\mathbb{R}\), and its range is \(\left(-\frac{\pi}{2},\frac{\pi}{2}\right)\).

The composition identities for \(\arctan\) are stated over the full real line on one side and the open principal interval on the other.

\begin{proposition}[Inverse tangent identities]
\begin{conditiondisplay}
\tan(\arctan x) &= x, \qquad & x\in\mathbb{R},\\
\arctan(\tan x) &= x, \qquad & -\frac{\pi}{2}<x<\frac{\pi}{2}.
\end{conditiondisplay}
\end{proposition}

\begin{workedexample}
\begin{example}
Simplify \(\cos(\arctan x)\).
\end{example}

\begin{solution}
Let \(y=\arctan x\). Then
\begin{conditiondisplay}
\tan y &= x, \qquad & -\frac{\pi}{2}<y<\frac{\pi}{2}.
\end{conditiondisplay}
We want to find \(\cos y\). Using \(1+\tan^{2}y=\sec^{2}y\), we get \(\sec^{2}y=1+x^{2}\). Since \(-\frac{\pi}{2}<y<\frac{\pi}{2}\), we have \(\cos y>0\) and hence \(\sec y>0\). Therefore \(\sec y=\sqrt{1+x^{2}}\), and so
\[
\cos(\arctan x)=\cos y=\frac{1}{\sec y}=\frac{1}{\sqrt{1+x^{2}}}. \qedhere
\]
\end{solution}
\end{workedexample}

\subsection{The remaining inverse trigonometric functions}

The remaining inverse trigonometric functions are used less frequently, but they are defined in the same spirit.

\begin{definition}
The \emph{inverse cosecant function}\index{inverse cosecant function}\index{arccosecant@$\arccsc$}, denoted by \(\arccsc x\), is defined by
\[
\arccsc x=y \quad \Longleftrightarrow \quad \csc y=x \ \text{and}\ y\in \left(0,\tfrac{\pi}{2}\right]\cup\left(\pi,\tfrac{3\pi}{2}\right].
\]
\end{definition}

\begin{definition}
The \emph{inverse secant function}\index{inverse secant function}\index{arcsecant@$\arcsec$}, denoted by \(\arcsec x\), is defined by
\[
\arcsec x=y \quad \Longleftrightarrow \quad \sec y=x \ \text{and}\ y\in \left[0,\tfrac{\pi}{2}\right)\cup\left[\pi,\tfrac{3\pi}{2}\right).
\]
\end{definition}

\begin{definition}
The \emph{inverse cotangent function}\index{inverse cotangent function}\index{arccotangent@$\arccot$}, denoted by \(\arccot x\), is defined by
\[
\arccot x=y \quad \Longleftrightarrow \quad \cot y=x \ \text{and}\ y\in (0,\pi).
\]
\end{definition}

The domains of \(\arccsc x\) and \(\arcsec x\) are \((-\infty,-1]\cup[1,\infty)\), while the domain of \(\arccot x\) is \(\mathbb{R}\). The principal ranges are the intervals specified in the definitions above.

\begin{remark}
For \(\arcsec x\), we can also choose \(y\in \left[0,\frac{\pi}{2}\right)\cup\left(\frac{\pi}{2},\pi\right]\). In this text we use \(y\in \left[0,\frac{\pi}{2}\right)\cup\left[\pi,\frac{3\pi}{2}\right)\) because later chapters will use derivatives of inverse trigonometric functions, and with this branch choice the differentiation formulas are simpler. When consulting other sources or software, always check which convention is being used, since different branch choices can lead to different values.
\end{remark}

\begin{remark}
For \(\arccsc x\), another common choice is \(y\in \left[-\frac{\pi}{2},0\right)\cup\left(0,\frac{\pi}{2}\right]\). In this text we use \(y\in \left(0,\frac{\pi}{2}\right]\cup\left(\pi,\frac{3\pi}{2}\right]\) so that the chosen branch parallels the branch selected for \(\arcsec x\). Once the principal range is fixed consistently, either convention gives a valid inverse function.
\end{remark}

\section{The Limit of a Function}

In many situations, we are interested not only in the value of a function at a point, but also in the behavior of the function when the input is close to that point. One important example comes from the study of motion.

Suppose that \(s(t)\) denotes the position of an object at time \(t\). Then the \emph{average velocity}\index{average velocity} of the object from time \(t\) to time \(t+h\) is
\[
\frac{s(t+h)-s(t)}{h}.
\]
More generally, for a function \(f\), the ratio
\[
\frac{f(y)-f(x)}{y-x}
\qquad \text{or} \qquad
\frac{f(x+h)-f(x)}{h}
\]
measures the \emph{average rate of change}\index{rate of change!average} of the function over an interval.

However, in many applications we do not want the average rate of change over an interval; we want the \emph{instantaneous rate of change}\index{rate of change!instantaneous}\index{instantaneous velocity} at a single moment. For example, in mechanics we distinguish between average velocity and instantaneous velocity. The average velocity describes the motion over a time interval, whereas the instantaneous velocity describes how fast the object is moving at one particular moment. To understand this instantaneous behavior, we let the second point move closer and closer to the first point. In other words, we study what happens when the input is very close to the point of interest. This leads naturally to the concept of the limit of a function.

\begin{definition}\label{def:limit-informal}
Suppose that \(f(x)\) is defined for all \(x\) near \(a\), except possibly at \(x=a\). We write
\[
\lim_{x\to a} f(x)=L
\]
and say that the \emph{limit}\index{limit}\index{limit@$\lim$} of \(f(x)\) as \(x\) approaches \(a\) is \(L\) if \(f(x)\) can be made arbitrarily close to \(L\) whenever \(x\) is sufficiently close to \(a\) but not equal to \(a\) \((x\ne a)\).
\end{definition}

This is an informal description; the phrases ``arbitrarily close'' and ``sufficiently close'' are intuitive but not yet rigorous. We will replace them with the precise \(\varepsilon\)-\(\delta\) formulation in \cref{def:limit-precise}.

When we consider \(\lim_{x\to a} f(x)\), we are concerned with the values of \(f(x)\) for \(x\) near \(a\), not necessarily with the value of \(f(a)\). In particular, \(f(a)\) need not be defined at all.

\bottomrule

\[
\lim_{x\to 1}\frac{x-1}{x^{2}-1}=\frac{1}{2}. \qedhere
\]
\end{solution}
\end{workedexample}

\[

\toprule

\midrule

These values are close to \(\frac{1}{6}=0.166666\cdots\), so we guess that
\[

\]
\end{solution}
\end{workedexample}

\section{One-Sided Limits, Infinite Limits, and Vertical Asymptotes}

In the previous section, we introduced the idea of the limit of a function. Sometimes, however, the behavior of a function from the left and from the right is different. This leads to the notion of one-sided limits. We also study situations in which the values of a function become arbitrarily large positive or arbitrarily large negative near a point.

\subsection{One-sided limits}

Near a point \(a\), a function can behave differently depending on which side \(x\) approaches from. A piecewise-defined function may switch formulas at \(a\); the graph may have a jump. In such cases we need to talk about the limit from the left and the limit from the right separately.

\begin{definition}
We write
\[
\lim_{x\to a^-}f(x)=L
\]
and say that the \emph{left-hand limit}\index{left-hand limit}\index{limit!left-hand} of \(f(x)\) as \(x\) approaches \(a\) is \(L\) if \(f(x)\) approaches \(L\) as \(x\) approaches \(a\) through values less than \(a\).

Similarly, we write
\[
\lim_{x\to a^+}f(x)=L
\]
and say that the \emph{right-hand limit}\index{right-hand limit}\index{limit!right-hand} of \(f(x)\) as \(x\) approaches \(a\) is \(L\) if \(f(x)\) approaches \(L\) as \(x\) approaches \(a\) through values greater than \(a\).
\end{definition}

\begin{proposition}[Criterion for a Two-Sided Limit]
\[
\lim_{x\to a}f(x)=L \quad \Longleftrightarrow \quad \lim_{x\to a^-}f(x)=L \ \text{and}\ \lim_{x\to a^+}f(x)=L.
\]
\end{proposition}

If the left-hand limit and the right-hand limit are different, then the two-sided limit does not exist.

\begin{workedexample}
\begin{example}
Let
\[
f(x)=
\begin{cases}
x+1, & x<2,\\[4pt]
4-x, & x\ge 2.
\end{cases}
\]
Evaluate \(\lim_{x\to 2^-}f(x)\), \(\lim_{x\to 2^+}f(x)\), and determine whether \(\lim_{x\to 2}f(x)\) exists.
\end{example}

\begin{solution}
From the left, we use the branch \(f(x)=x+1\), so \(\lim_{x\to 2^-}f(x)=3\).

From the right, we use the branch \(f(x)=4-x\), so \(\lim_{x\to 2^+}f(x)=2\).

Since the left-hand limit and the right-hand limit are different, \(\lim_{x\to 2}f(x)\) does not exist.
\end{solution}
\end{workedexample}

\subsection{Infinite limits}

In some cases, the values of \(f(x)\) do not approach a finite number as \(x\to a\). Instead, they become arbitrarily large positive or arbitrarily large negative.

\begin{definition}\label{def:infinite-limit-informal}
Let \(f\) be a function defined on both sides of \(a\), except possibly at \(a\) itself. We write
\[
\lim_{x\to a}f(x)=\infty
\]
if the values of \(f(x)\) can be made arbitrarily large by taking \(x\) sufficiently close to \(a\), with \(x\ne a\). In this case we call \(\lim_{x\to a}f(x)\) an \emph{infinite limit}\index{infinite limit}\index{limit!infinite}.

Similarly, we write
\[
\lim_{x\to a}f(x)=-\infty
\]
if the values of \(f(x)\) can be made arbitrarily large negative by taking \(x\) sufficiently close to \(a\), with \(x\ne a\).
\end{definition}

This is again an informal description; a precise \(\varepsilon\)-\(\delta\)-style formulation is given in \cref{def:infinite-limit-precise}.

The symbols \(\infty\) and \(-\infty\) do not represent real numbers. They indicate that a function grows without bound (positively or negatively) near a specific point.

Similar definitions apply to one-sided infinite limits:
\[
\lim_{x\to a^-}f(x)=\infty,
\qquad
\lim_{x\to a^+}f(x)=\infty,
\qquad
\lim_{x\to a^-}f(x)=-\infty,
\qquad
\lim_{x\to a^+}f(x)=-\infty.
\]
The four possibilities are illustrated in \cref{fig:one-sided-infinite-limits}.

Consider the function

\frac{1}{(0.1)^2}=100,
\qquad

\end{solution}

\subsection{Vertical asymptotes}

Infinite limits are closely related to vertical asymptotes.

\begin{definition}
The vertical line \(x=a\) is called a \emph{vertical asymptote}\index{vertical asymptote}\index{asymptote!vertical} of the curve \(y=f(x)\) if at least one of the following statements is true:
\[
\lim_{x\to a^-}f(x)=\infty,
\qquad
\lim_{x\to a^+}f(x)=\infty,
\qquad
\lim_{x\to a^-}f(x)=-\infty,
\qquad
\lim_{x\to a^+}f(x)=-\infty.
\]
\end{definition}

If \(x=a\) is a vertical asymptote, then the function values become arbitrarily large positive or arbitrarily large negative as \(x\) approaches \(a\) from at least one side.

\begin{workedexample}
\begin{example}\index{1/x near 0@$1/x$ near $0$}
Evaluate \(\lim_{x\to 0^-}\frac{1}{x}\) and \(\lim_{x\to 0^+}\frac{1}{x}\), determine whether \(\lim_{x\to 0}\frac{1}{x}\) exists, and decide whether \(x=0\) is a vertical asymptote of the graph of \(y=\frac{1}{x}\).
\end{example}

\begin{solution}
As \(x\to 0^{-}\), the values of \(x\) are negative and very close to \(0\), so \(\frac{1}{x}\) is a large negative number. Hence \(\lim_{x\to 0^-}\frac{1}{x}=-\infty\).

As \(x\to 0^{+}\), the values of \(x\) are positive and very close to \(0\), so \(\frac{1}{x}\) is a large positive number. Hence \(\lim_{x\to 0^+}\frac{1}{x}=\infty\).

Since the left-hand and right-hand limits are different, the two-sided limit \(\lim_{x\to 0}\frac{1}{x}\) does not exist. However, at least one one-sided infinite limit occurs, so the line \(x=0\) is a vertical asymptote of the graph of \(y=\frac{1}{x}\).
\end{solution}
\end{workedexample}

\begin{workedexample}
\begin{example}
Does the curve \(y=\frac{2x}{x-3}\) have a vertical asymptote?
\end{example}

\begin{solution}
The denominator is zero when \(x=3\), so \(x=3\) is the only possible vertical asymptote.

As \(x\to 3^{+}\), the denominator \(x-3\) is a small positive number while \(2x\) is close to \(6\). Therefore \(\frac{2x}{x-3}\) becomes arbitrarily large positive, so \(\lim_{x\to 3^+}\frac{2x}{x-3}=\infty\).

As \(x\to 3^{-}\), the denominator \(x-3\) is a small negative number while \(2x\) is still close to \(6\). Therefore \(\frac{2x}{x-3}\) becomes arbitrarily large negative, so \(\lim_{x\to 3^-}\frac{2x}{x-3}=-\infty\).

Therefore the line \(x=3\) is a vertical asymptote of the curve, as shown in \cref{fig:vertical-asymptote-example}.
\end{solution}
\end{workedexample}

\end{workedexample}

A two-sided infinite limit such as \(\lim_{x\to a}f(x)=\infty\) requires the function to approach \(\infty\) from both sides. If the behavior from the left and from the right is different, then the two-sided limit does not exist, even though \(x=a\) may still be a vertical asymptote.

\section{Limit Laws and Computational Techniques}

In the previous sections, we used tables and graphs to guess the values of limits. We now introduce algebraic tools for evaluating limits more systematically. The basic idea is that limits behave well with respect to the usual algebraic operations.

\subsection{The limit laws}

\begin{theorem}[Limit laws]\index{limit laws}
Suppose that \(c\) is a constant and that both \(\lim_{x\to a}f(x)\) and \(\lim_{x\to a}g(x)\) exist. Then:
\begin{aligneddisplay}
\lim_{x\to a}\bigl(f(x)+g(x)\bigr)
&=
\lim_{x\to a}f(x)+\lim_{x\to a}g(x),\\
\lim_{x\to a}\bigl(f(x)-g(x)\bigr)
&=
\lim_{x\to a}f(x)-\lim_{x\to a}g(x),\\
\lim_{x\to a}\bigl(c f(x)\bigr)
&=
c\lim_{x\to a}f(x),\\
\lim_{x\to a}\bigl(f(x)g(x)\bigr)
&=
\left(\lim_{x\to a}f(x)\right)\left(\lim_{x\to a}g(x)\right).
\end{aligneddisplay}
If \(\lim_{x\to a}g(x)\ne 0\), then
\[
\lim_{x\to a}\frac{f(x)}{g(x)}
=
\frac{\lim_{x\to a}f(x)}{\lim_{x\to a}g(x)}.
\]
\end{theorem}

These laws can be summarized verbally as follows:
\begin{enumerate}
    \item The limit of a sum is the sum of the limits.
    \item The limit of a difference is the difference of the limits.
    \item The limit of a constant multiple is the constant times the limit.
    \item The limit of a product is the product of the limits.
    \item The limit of a quotient is the quotient of the limits, provided the denominator limit is not \(0\).
\end{enumerate}

These properties will be proved rigorously later. For now, they provide an effective way to evaluate many limits.

\subsection{Direct substitution}

The limit laws imply several useful consequences.

\begin{proposition}
Let \(n\) be a positive integer. Then
\begin{aligneddisplay}
\lim_{x\to a}x &= a,\\
\lim_{x\to a}x^{n} &= a^{n}.
\end{aligneddisplay}
Consequently, if \(p(x)\) is a polynomial, then
\[
\lim_{x\to a}p(x)=p(a).
\]
If \(r(x)\) is a rational function and \(r(a)\) is defined, then
\[
\lim_{x\to a}r(x)=r(a).
\]
\end{proposition}

So for polynomials and rational functions, the first step is usually direct substitution. If substitution gives a meaningful number, then the limit is found immediately.

\subsection{Algebraic simplification}

Sometimes direct substitution does not work because it leads to an expression such as \(\frac{0}{0}\). In that case, we try to simplify the expression first.

\begin{workedexample}
\begin{example}
Evaluate
\[
\lim_{x\to 1}\frac{x-1}{x^{2}-1}.
\]
\end{example}

\begin{solution}
Direct substitution gives
\[
\frac{1-1}{1^{2}-1}=\frac{0}{0},
\]
so we first simplify the expression:
\begin{aligneddisplay}
\frac{x-1}{x^{2}-1}
&=
\frac{x-1}{(x-1)(x+1)}\\
&=
\frac{1}{x+1},
\qquad x\ne 1.
\end{aligneddisplay}
Therefore, \(\lim_{x\to 1}\frac{x-1}{x^{2}-1}=\lim_{x\to 1}\frac{1}{x+1}=\frac{1}{2}\).
\end{solution}
\end{workedexample}

\begin{workedexample}
\begin{example}
Evaluate
\[
\lim_{t\to 0}\frac{\sqrt{t^{2}+9}-3}{t^{2}}.
\]
\end{example}

\begin{solution}
Direct substitution gives the indeterminate form \(\frac{0}{0}\), so we rationalize the numerator:
\begin{aligneddisplay}
\frac{\sqrt{t^{2}+9}-3}{t^{2}}
\cdot
\frac{\sqrt{t^{2}+9}+3}{\sqrt{t^{2}+9}+3}
&=
\frac{t^{2}+9-9}{t^{2}\bigl(\sqrt{t^{2}+9}+3\bigr)}\\
&=
\frac{1}{\sqrt{t^{2}+9}+3},
\qquad t\ne 0.
\end{aligneddisplay}
Now we may take the limit directly and obtain \(\lim_{t\to 0}\frac{\sqrt{t^{2}+9}-3}{t^{2}}=\lim_{t\to 0}\frac{1}{\sqrt{t^{2}+9}+3}=\frac{1}{6}\).
\end{solution}
\end{workedexample}

\subsection{Limits of piecewise-defined functions}

Not all limits can be evaluated by direct substitution alone. Sometimes we must use the definition of the function more carefully.

\begin{workedexample}
\begin{example}
Evaluate \(\lim_{x\to 0}|x|\).
\end{example}

\begin{solution}
Using
\[
|x|=
\begin{cases}
x, & x\ge 0,\\[4pt]
-x, & x<0.
\end{cases}
\]
we see that if \(x\to 0^{+}\), then \(|x|=x\to 0\), while if \(x\to 0^{-}\), then \(|x|=-x\to 0\). Therefore, \(\lim_{x\to 0}|x|=0\).
\end{solution}
\end{workedexample}

\begin{workedexample}
\begin{example}
The greatest integer function, also called the floor function, is denoted by \(\lfloor x \rfloor\) and defined to be the largest integer less than or equal to \(x\). Evaluate \(\lim_{x\to 3^{+}}\lfloor x \rfloor\) and \(\lim_{x\to 3^{-}}\lfloor x \rfloor\).
\end{example}

\begin{solution}
As \(x\to 3^{+}\), we have \(\lfloor x \rfloor=3\), so \(\lim_{x\to 3^{+}}\lfloor x \rfloor=3\).

As \(x\to 3^{-}\), we have \(\lfloor x \rfloor=2\), so \(\lim_{x\to 3^{-}}\lfloor x \rfloor=2\).

Since the left-hand and right-hand limits are different, we conclude that \(\lim_{x\to 3}\lfloor x \rfloor\) does not exist.
\end{solution}
\end{workedexample}

\subsection{The squeeze theorem}

A very useful technique for evaluating limits is to compare a function with two simpler functions.

\begin{theorem}[Squeeze theorem]\index{squeeze theorem}
Suppose that
\[
g(x)\le f(x)\le h(x)
\]
for all \(x\) near \(a\), except possibly at \(x=a\). If
\pairdisplay{\lim_{x\to a}g(x)=L,}{\lim_{x\to a}h(x)=L,}
then
\[
\lim_{x\to a}f(x)=L.
\]
\end{theorem}

\begin{workedexample}
\begin{example}\index{x squared sin(1/x) example@$x^{2}\sin(1/x)$ example}
Evaluate
\[
\lim_{x\to 0}x^{2}\sin\left(\frac{1}{x}\right).
\]
\end{example}

\begin{solution}
Because
\[
-1\le \sin\left(\frac{1}{x}\right)\le 1,
\]
for all \(x\ne 0\), we may multiply all parts by \(x^{2}\ge 0\) to obtain
\[
-x^{2}\le x^{2}\sin\left(\frac{1}{x}\right)\le x^{2}.
\]
Also,
\pairdisplay{\lim_{x\to 0}(-x^{2})=0,}{\lim_{x\to 0}x^{2}=0.}
Therefore, by the squeeze theorem,
\[
\lim_{x\to 0}x^{2}\sin\left(\frac{1}{x}\right)=0. \qedhere
\]
\end{solution}
\end{workedexample}

\begin{workedexample}
\begin{example}
Evaluate
\[
\lim_{x\to 0}x\cos\left(\frac{1}{x}\right).
\]
\end{example}

\begin{solution}
Because
\[
\left|\cos\left(\frac{1}{x}\right)\right|\le 1,
\]
for all \(x\ne 0\), we have
\[
\left|x\cos\left(\frac{1}{x}\right)\right|
\le |x|.
\]
Equivalently,
\[
-|x|\le x\cos\left(\frac{1}{x}\right)\le |x|.
\]
Also,
\pairdisplay{\lim_{x\to 0}(-|x|)=0,}{\lim_{x\to 0}|x|=0.}
Therefore, by the squeeze theorem,
\[
\lim_{x\to 0}x\cos\left(\frac{1}{x}\right)=0. \qedhere
\]
\end{solution}
\end{workedexample}

The limit laws and the techniques of factoring, rationalization, and squeezing cover most of the limits that appear at this level. Before moving on, it is worth recording the workflow in one place.

\begin{strategy}[Computing a limit \(\lim_{x\to a} f(x)\)]
\begin{enumerate}
    \item \emph{Try direct substitution first.} If \(f(a)\) is defined and the expression is continuous at \(a\), then the limit equals \(f(a)\).
    \item \emph{If substitution gives an indeterminate form} such as \(\tfrac{0}{0}\), simplify algebraically: factor, cancel common factors, rationalize numerator or denominator, or combine fractions.
    \item \emph{For a piecewise-defined function}, evaluate the one-sided limits separately using the branch that applies on each side, and check whether they agree.
    \item \emph{If the expression is trapped between two simpler ones} whose limits are known and equal, apply the squeeze theorem.
    \item \emph{If the values of the function grow without bound} as \(x\to a\), the limit is \(\infty\) or \(-\infty\) and the line \(x=a\) is a vertical asymptote.
\end{enumerate}
\end{strategy}

In the next section, we will give the precise definition of a limit.

\section{The Precise Definition of a Limit}

From the time of Newton and Leibniz, calculus was remarkably successful in solving problems in geometry, mechanics, and astronomy. However, for a long time, there was also controversy about its logical foundations. Early arguments often relied on infinitesimals---quantities thought of as being extremely small, but not described in a fully precise way. As a result, mathematicians continued to ask what it really means to say that \(x\) approaches \(a\) or that \(f(x)\) approaches \(L\).

In the previous sections, we used graphs, tables, and algebraic techniques to study limits. However, informal expressions such as ``\(x\) is close to \(a\)'' and ``\(f(x)\) is close to \(L\)'' are not rigorous enough. The modern resolution to this difficulty is the \(\varepsilon\)-\(\delta\) definition\index{epsilon-delta definition@$\varepsilon$-$\delta$ definition} of a limit, developed in the nineteenth century, which gave calculus the level of rigor used in modern mathematics. Instead of relying on vague language, it describes exactly what it means for \(f(x)\) to be close to \(L\) whenever \(x\) is sufficiently close to \(a\). We now introduce this precise definition.

To motivate the definition, consider the function
\[
f(x)=
\begin{cases}
2x-1, & x\ne 3,\\[4pt]
6, & x=3.
\end{cases}
\]
When \(x\) is close to \(3\), but \(x\ne 3\), the value of \(f(x)=2x-1\) is close to \(5\). Thus it is natural to expect that
\(
\lim_{x\to 3} f(x)=5.
\)

\begin{workedexample}
\begin{example}
For the function
\[
f(x)=
\begin{cases}
2x-1, & x\ne 3,\\[4pt]
6, & x=3,
\end{cases}
\]
how close must \(x\) be to \(3\) in order to guarantee that \(f(x)\) differs from \(5\) by less than \(0.1\)?
\end{example}

\begin{solution}
We want \(|f(x)-5|<0.1\). For \(x\ne 3\), we have \(f(x)=2x-1\), so
\begin{aligneddisplay}
|f(x)-5|
&=
|(2x-1)-5|\\
&=
|2x-6|\\
&=
2|x-3|.
\end{aligneddisplay}
Thus it is enough to require \(2|x-3|<0.1\), which is equivalent to \(|x-3|<0.05\). Therefore, if \(0<|x-3|<0.05\), then \(f(x)\) is within \(0.1\) of \(5\).
\end{solution}
\end{workedexample}

If we replace \(0.1\) by the smaller number \(0.01\), then the same reasoning shows that \(|f(x)-5|<0.01\) whenever \(0<|x-3|<0.005\).
Thus, when we ask for \(f(x)\) to be closer to \(5\), we must require \(x\) to be closer to \(3\).

More generally, if we want \(|f(x)-5|<\varepsilon\) for some \(\varepsilon>0\), then it is enough to require \(2|x-3|<\varepsilon\), equivalently \(|x-3|<\frac{\varepsilon}{2}\). Thus for this example we may choose \(\delta=\frac{\varepsilon}{2}\).

The previous discussion shows the basic idea. We want to control the distance between \(f(x)\) and the limit \(L\) by controlling the distance between \(x\) and the point \(a\).

\begin{definition}\label{def:limit-precise}
Let \(f\) be a function defined on an open interval containing \(a\), except possibly at \(a\) itself. We say that
\(
\lim_{x\to a} f(x)=L
\)
if for every \(\varepsilon>0\) there exists a number \(\delta>0\) such that
\[
|f(x)-L|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-a|<\delta.
\]
\emph{Informally, no matter how small a tolerance \(\varepsilon\) we demand for \(f(x)\) to be within of \(L\), there is always some distance \(\delta\) from \(a\) such that every \(x\) within that distance (and not equal to \(a\)) forces \(f(x)\) to land inside the tolerance.}
\end{definition}

This is the rigorous counterpart of the informal \cref{def:limit-informal}: ``\(f(x)\) is arbitrarily close to \(L\)'' becomes ``\(|f(x)-L|<\varepsilon\) for every \(\varepsilon>0\),'' and ``\(x\) is sufficiently close to \(a\), but \(x\ne a\)'' becomes ``\(0<|x-a|<\delta\) for a suitably chosen \(\delta\).''

\Cref{fig:precise-limit} illustrates the definition. If \(x\) is restricted to lie in the interval
\((a-\delta,a+\delta)\), \(x\ne a\),
then the corresponding values of \(f(x)\) are forced to lie in the interval
\((L-\varepsilon,L+\varepsilon)\).
Notice that the value \(f(a)\) itself may be different from \(L\). The limit depends only on the behavior of \(f(x)\) when \(x\) is close to \(a\), not necessarily on the value of the function at \(x=a\).

The condition \( 0<|x-a|<\delta \) means that \(x\) is within a distance \(\delta\) of \(a\), but \(x\ne a\). The inequality \( |f(x)-L|<\varepsilon \) means that \(f(x)\) is within a distance \(\varepsilon\) of \(L\). Moreover, the statement \( \lim_{x\to a} f(x)=L \) must hold for every \(\varepsilon>0\), no matter how small \(\varepsilon\) is.

\begin{proposition}[Uniqueness of Limits]
If \(\lim_{x\to a} f(x)\) exists, then it is unique.
\end{proposition}

\begin{proof}
Suppose that
\begin{aligneddisplay}
\lim_{x\to a} f(x) &= L,\\
\lim_{x\to a} f(x) &= M.
\end{aligneddisplay}
If \(L\ne M\), let
\[
\varepsilon=\frac{|L-M|}{2}.
\]
Then there exist \(\delta_1,\delta_2>0\) such that
\[
|f(x)-L|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-a|<\delta_1,
\]
and
\[
|f(x)-M|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-a|<\delta_2.
\]
If \(0<|x-a|<\min\{\delta_1,\delta_2\}\), then both inequalities hold. By the triangle inequality,
\begin{aligneddisplay}
|L-M|
&=
|L-f(x)+f(x)-M|\\
&\le
|L-f(x)|+|f(x)-M|\\
&<
2\varepsilon\\
&=
|L-M|,
\end{aligneddisplay}
which is impossible. Therefore \(L=M\).
\end{proof}

\begin{strategy}[Verifying a limit from the \(\varepsilon\)-\(\delta\) definition]
To show that \(\lim_{x\to a}f(x)=L\):
\begin{enumerate}
    \item Start with a generic \(\varepsilon>0\). Our task is to find a \(\delta>0\) (which may depend on \(\varepsilon\)) that makes the definition work.
    \item Work with the quantity \(|f(x)-L|\) and simplify or bound it by an expression that involves \(|x-a|\).
    \item Choose \(\delta\) in terms of \(\varepsilon\) so that the bound above is less than \(\varepsilon\) whenever \(|x-a|<\delta\). When the simplification produces an auxiliary factor that depends on \(x\), first restrict \(x\) to a small neighborhood (say \(|x-a|<1\)) to bound that factor, and then take \(\delta\) to be the minimum of \(1\) and the \(\varepsilon\)-based bound.
    \item Verify the definition: show that if \(0<|x-a|<\delta\), then \(|f(x)-L|<\varepsilon\).
\end{enumerate}
\end{strategy}

\begin{workedexample}
\begin{example}
Show from the definition that \(\lim_{x\to 3} (4x-5)=7\).
\end{example}

\begin{solution}
Given \(\varepsilon>0\), we want to find \(\delta>0\) such that
\[
|(4x-5)-7|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-3|<\delta.
\]
Since
\begin{aligneddisplay}
|(4x-5)-7|
&=
|4x-12|\\
&=
4|x-3|.
\end{aligneddisplay}
Thus it is enough to require \(4|x-3|<\varepsilon\), which holds whenever \(|x-3|<\dfrac{\varepsilon}{4}\). We therefore choose \(\delta=\frac{\varepsilon}{4}\).
Then, whenever \(0<|x-3|<\delta\), we have
\begin{aligneddisplay}
|(4x-5)-7|
&=
4|x-3|\\
&<
4\left(\frac{\varepsilon}{4}\right)\\
&=
\varepsilon.
\end{aligneddisplay}
Therefore, \(\lim_{x\to 3} (4x-5)=7\).
\end{solution}
\end{workedexample}

\begin{workedexample}
\begin{example}
Show from the definition that \(\lim_{x\to 1} (x^{2}-5x+6)=2\).
\end{example}

\begin{solution}
Given \(\varepsilon>0\), we want to find \(\delta>0\) such that
\[
|(x^{2}-5x+6)-2|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-1|<\delta.
\]
First we simplify:
\begin{aligneddisplay}
|(x^{2}-5x+6)-2|
&=
|x^{2}-5x+4|\\
&=
|(x-1)(x-4)|.
\end{aligneddisplay}
The difficulty is that the factor \(|x-4|\) also depends on \(x\). To control it, we first assume \(|x-1|<1\). Then
\begin{aligneddisplay}
|x-1|<1
&\implies -1<x-1<1 \\
&\implies 0<x<2 \\
&\implies -4<x-4<-2 \\
&\implies |x-4|<4.
\end{aligneddisplay}
Thus, whenever \(|x-1|<1\), we have
\[
|(x-1)(x-4)|<4|x-1|.
\]
To guarantee that \(|(x-1)(x-4)|<\varepsilon\), it is therefore enough to require both \(|x-1|<1\) and \(4|x-1|<\varepsilon\). We choose
\[
\delta=\min\left\{1,\frac{\varepsilon}{4}\right\}.
\]
Then, whenever \(0<|x-1|<\delta\), we have both \(|x-1|<1\) and \(|x-1|<\dfrac{\varepsilon}{4}\), so \(|x-4|<4\) and
\begin{aligneddisplay}
|(x^{2}-5x+6)-2|
&=
|(x-1)(x-4)|\\
&<
4|x-1|\\
&<
4\left(\frac{\varepsilon}{4}\right)\\
&=
\varepsilon.
\end{aligneddisplay}
Therefore, \(\lim_{x\to 1} (x^{2}-5x+6)=2\).
\end{solution}
\end{workedexample}

\begin{definition}\label{def:infinite-limit-precise}
We write
\(
\lim_{x\to a} f(x)=\infty
\)
if for every positive number \(M\), there exists \(\delta>0\) such that
\[
f(x)>M
\qquad \text{whenever} \qquad
0<|x-a|<\delta.
\]
Similarly, we write
\(
\lim_{x\to a} f(x)=-\infty
\)
if for every negative number \(N\), there exists \(\delta>0\) such that
\[
f(x)<N
\qquad \text{whenever} \qquad
0<|x-a|<\delta.
\]
\end{definition}

This formalizes the informal notion introduced in \cref{def:infinite-limit-informal}: ``\(f(x)\) can be made arbitrarily large'' becomes ``\(f(x)>M\) for every \(M>0\),'' and the analogous replacement applies for \(-\infty\). We may think of \(M\) as any large positive threshold, such as \(100\), \(1000\), \(10000\), and so on: no matter how large a bound \(M\) we fix, \(f(x)\) eventually exceeds it whenever \(x\) is close enough to \(a\). Similarly, \(N\) is any negative threshold, such as \(-100\), \(-1000\), \(-10000\), and so on, below which \(f(x)\) is eventually forced.

The \(\varepsilon\)-\(\delta\) formulation also allows us to give a precise meaning to a property of functions that will appear throughout the chapters to come. If \(f\) is defined at every point near \(a\), including \(a\), and if
\[
\lim_{x\to a} f(x)=f(a),
\]
then \(f\) is said to be \emph{continuous at \(a\)}\index{continuous function}\index{continuity}. We mention this term here only because it is expressed directly in terms of limits; a systematic study of continuity, together with its main properties and theorems, belongs to a later chapter.

\section*{Summary}

\paragraph{Key definitions.}
\begin{itemize}
    \item \emph{One-to-one function} --- different inputs always produce different outputs.
    \item \emph{Inverse function} \(f^{-1}\) --- reverses the action of a one-to-one function \(f\).
    \item \emph{Inverse trigonometric functions} \(\arcsin\), \(\arccos\), \(\arctan\), \(\arccsc\), \(\arcsec\), \(\arccot\) --- defined by restricting each trigonometric function to a principal interval on which it is one-to-one.
    \item \emph{Limit} \(\lim_{x\to a}f(x)=L\) --- the values of \(f\) are arbitrarily close to \(L\) when \(x\) is sufficiently close to but not equal to \(a\) (informal); for every \(\varepsilon>0\) there exists \(\delta>0\) such that \(|f(x)-L|<\varepsilon\) whenever \(0<|x-a|<\delta\) (precise).
    \item \emph{One-sided limits} \(\lim_{x\to a^-}f(x)\) and \(\lim_{x\to a^+}f(x)\) --- restrict the approach to values less than or greater than \(a\), respectively.
    \item \emph{Infinite limits} \(\lim_{x\to a}f(x)=\pm\infty\) --- the values of \(f\) grow without bound as \(x\) approaches \(a\).
    \item \emph{Vertical asymptote} at \(x=a\) --- at least one of the one-sided limits at \(a\) is infinite.
    \item \emph{Continuity at a point}, introduced in terms of limits as \(\lim_{x\to a}f(x)=f(a)\); studied systematically in a later chapter.
\end{itemize}

\paragraph{Key theorems, propositions, and corollaries.}
\begin{itemize}
    \item \emph{Horizontal line test} --- \(f\) is one-to-one if and only if every horizontal line meets its graph at most once.
    \item \emph{Existence of an inverse} --- a function has an inverse if and only if it is one-to-one.
    \item \emph{Inverse trigonometric identities} --- on the principal ranges, \(\sin(\arcsin x)=x\) and \(\arcsin(\sin x)=x\), and analogous identities for \(\arccos\) and \(\arctan\).
    \item \emph{Criterion for a two-sided limit} --- \(\lim_{x\to a}f(x)=L\) if and only if both one-sided limits at \(a\) equal \(L\).
    \item \emph{Limit laws} --- limits respect sums, differences, products, quotients (with non-zero denominator limit), and powers.
    \item \emph{Squeeze theorem} --- if \(g(x)\le f(x)\le h(x)\) near \(a\) and \(g\) and \(h\) share a common limit \(L\) at \(a\), then so does \(f\).
    \item \emph{Uniqueness of limits} --- a limit, if it exists, is unique.
\end{itemize}

\paragraph{Key formulas and identities.}
\begin{itemize}
    \item Inverse composition: \(f^{-1}(f(x))=x\) for \(x\) in the domain of \(f\), and \(f(f^{-1}(y))=y\) for \(y\) in the range of \(f\).
    \item Principal ranges of the inverse trig functions: \(\arcsin x\in\left[-\tfrac{\pi}{2},\tfrac{\pi}{2}\right]\), \(\arccos x\in[0,\pi]\), \(\arctan x\in\left(-\tfrac{\pi}{2},\tfrac{\pi}{2}\right)\).
    \item Direct substitution: for polynomials and rational functions, \(\lim_{x\to a}p(x)=p(a)\) where defined.
    \item Two-sided-limit criterion: \(\lim_{x\to a}f(x)=L\) exactly when both one-sided limits \(\lim_{x\to a^\pm}f(x)\) equal \(L\).
    \item Precise \(\varepsilon\)-\(\delta\) statement: \(\lim_{x\to a}f(x)=L\) means that for every \(\varepsilon>0\) there exists \(\delta>0\) such that \(|f(x)-L|<\varepsilon\) whenever \(0<|x-a|<\delta\).
\end{itemize}
