\chapter{Inverse Functions and Limits}

This chapter develops two themes that together form the starting point of calculus. The first is the idea of reversing a function: when a process sends each input to a distinct output, we can ask how to recover the input from the output. This leads to the notions of one-to-one functions and inverse functions, and in particular to the inverse trigonometric functions, which appear throughout geometry, physics, and later differentiation formulas.

The second theme is the limit of a function. Many quantities of interest in calculus, such as the instantaneous rate of change of a moving object, cannot be read off from a single value of a function; they are determined by the behavior of the function near a point. We first study limits informally by inspecting tables and graphs, extend the idea to one-sided and infinite limits together with their geometric counterpart---vertical asymptotes---and then develop algebraic techniques for computing limits. The chapter closes with the \(\varepsilon\)-\(\delta\) definition, which places all of the preceding discussion on a rigorous foundation and prepares the way for derivatives and integrals in the chapters that follow.

\section{Inverse Functions}

In many situations, a function describes a process that sends an input \(x\) to an output \(y=f(x)\). We are often interested in reversing this process and recovering the original input from the output. This leads to the ideas of one-to-one functions and inverse functions. For example, if a formula converts temperature from Celsius to Fahrenheit, then the inverse formula converts Fahrenheit back to Celsius. In the same way, many scientific formulas describe a process in one direction, and an inverse function allows us to recover the original quantity.

\subsection{One-to-one functions}

A function can have an inverse only when each output comes from exactly one input. This motivates the following definition.

\begin{definition}
Let \(f\) be a function with domain \(A\). We say that \(f\) is \emph{one-to-one}\index{one-to-one function} if
\[
f(x_1)\ne f(x_2)
\qquad \text{whenever } x_1\ne x_2.
\]
Equivalently, \(f\) is one-to-one if
\[
f(x_1)=f(x_2) \implies x_1=x_2.
\]
\end{definition}

\begin{example}
Consider the following functions defined on a group of students:
\[
f(\text{student})=\text{Student ID number}
\]
and
\[
g(\text{student})=\text{Blood type}.
\]
Determine which of these real-world functions are one-to-one.
\end{example}

\begin{solution}
The function \(f\) is one-to-one because every student gets a unique ID number. Distinct inputs always give distinct outputs, so each ID number corresponds to exactly one student.

The function \(g\) is not one-to-one because different students can share the same blood type. For instance, if Alice and Bob are both Type O, then
\[
g(\text{Alice})=\text{Type O}
\qquad \text{and} \qquad
g(\text{Bob})=\text{Type O}.
\]
Since two different inputs produce the same output, \(g\) is not one-to-one.
\end{solution}

Now let us consider some mathematical examples.

\begin{example}
Consider the functions
\[
f(x)=x, \qquad 0\le x\le 1,
\]
and
\[
g(x)=x^2, \qquad -1\le x\le 1.
\]
Determine which of these functions are one-to-one.
\end{example}

\begin{solution}
The function \(f(x)=x\) is one-to-one on \([0,1]\) because distinct inputs always produce distinct outputs.

The function \(g(x)=x^2\) is not one-to-one on \([-1,1]\) because
\[
g\left(\frac12\right)=\left(\frac12\right)^2=\frac14
\qquad \text{and} \qquad
g\left(-\frac12\right)=\left(-\frac12\right)^2=\frac14.
\]
Thus two different inputs produce the same output, so \(g\) is not one-to-one.
\end{solution}

Not every function has an inverse. A function can have an inverse only if each output corresponds to exactly one input; in other words, the function must be one-to-one. For \(g(x)=x^2\) on \([-1,1]\), the value \(\frac14\) comes from both \(\frac12\) and \(-\frac12\). Therefore \(g^{-1}\!\left(\frac14\right)\) is not well defined.

\begin{remark}[Horizontal Line Test]\index{horizontal line test}
A function is one-to-one if and only if no horizontal line intersects its graph more than once. A horizontal line has equation \(y=c\), so it represents a fixed output value \(c\). If the line \(y=c\) intersects the graph of \(f\) at two distinct points \((x_1,c)\) and \((x_2,c)\) with \(x_1\ne x_2\), then
\[
f(x_1)=c=f(x_2).
\]
Thus two different inputs produce the same output, so \(f\) is not one-to-one. Conversely, if every horizontal line intersects the graph at most once, then no output value corresponds to more than one input, and hence \(f\) is one-to-one. This criterion is illustrated in \cref{fig:horizontal-line-test}.
\end{remark}

\subsection{Inverse functions}

Suppose that \(f\) is a one-to-one function. If \(f\) maps \(x\) to \(y\), then its inverse maps \(y\) back to \(x\).

\begin{definition}
Let \(f\) be a one-to-one function with domain \(A\) and range \(B\). Then the \emph{inverse function}\index{inverse function}\index{inverse function@$f^{-1}$} of \(f\), denoted by \(f^{-1}\), has domain \(B\) and range \(A\), and is defined by
\[
f^{-1}(y)=x
\quad \Longleftrightarrow \quad
f(x)=y,
\qquad y\in B.
\]
\end{definition}

When we focus on the inverse function, it is common to use \(x\) as the independent variable. In that case we may write
\[
f^{-1}(x)=y
\quad \Longleftrightarrow \quad
f(y)=x.
\]

\begin{theorem}
A function has an inverse function if and only if it is one-to-one.
\end{theorem}

\begin{proof}
Suppose first that \(f\) has an inverse function \(f^{-1}\). If \(f(x_1)=f(x_2)\), then applying \(f^{-1}\) to both sides gives
\[
f^{-1}(f(x_1))=f^{-1}(f(x_2)),
\]
so \(x_1=x_2\). Hence \(f\) is one-to-one.

Conversely, suppose that \(f\) is one-to-one with domain \(A\) and range \(B\). For each \(y\in B\), there is some \(x\in A\) such that \(f(x)=y\), and the one-to-one property guarantees that this \(x\) is unique. Therefore the correspondence \(y\mapsto x\) defines an inverse function \(f^{-1}:B\to A\).
\end{proof}

\begin{example}
Find the inverse of each function.
\begin{enumerate}
    \item \(f(x)=x\)
    \item \(g(x)=x^3\)
\end{enumerate}
\end{example}

\begin{solution}
\begin{enumerate}
    \item  Since \(f(x)=x\) leaves every input unchanged, its inverse is also
    \[
    f^{-1}(x)=x.
    \]

    \item Let
    \[
    y=x^3.
    \]
    Solving for \(x\), we obtain
    \[
    x=\sqrt[3]{y}.
    \]
    Interchanging \(x\) and \(y\), we get
    \[
    g^{-1}(x)=\sqrt[3]{x}.
    \]
\end{enumerate}
\end{solution}

\begin{proposition}
Let \(f\) be a one-to-one function with domain \(A\) and range \(B\). Then
\[
f^{-1}(f(x))=x
\qquad \text{for every } x\in A,
\]
and
\[
f(f^{-1}(x))=x
\qquad \text{for every } x\in B.
\]
\end{proposition}

\subsection{Finding the inverse of a function}

When \(f\) is one-to-one, the following procedure is often useful for finding \(f^{-1}\):
\begin{enumerate}
    \item Write \(y=f(x)\).
    \item Solve this equation for \(x\) in terms of \(y\).
    \item Interchange \(x\) and \(y\). The resulting equation gives \(f^{-1}(x)\).
\end{enumerate}

\begin{example}
Find the inverse of
\(
f(x)=x^3+2.
\)
\end{example}

\begin{solution}
Let
\[
y=x^3+2.
\]
Then
\[
x^3=y-2,
\]
so
\[
x=\sqrt[3]{\,y-2\,}.
\]
Interchanging \(x\) and \(y\), we obtain
\[
f^{-1}(x)=\sqrt[3]{\,x-2\,}.
\]
We check this by composition:
\[
f\bigl(f^{-1}(x)\bigr)
=
\left(\sqrt[3]{\,x-2\,}\right)^3+2
=
x-2+2
=
x.
\]
Thus the inverse is correct.
\end{solution}

\begin{example}
Find the inverse of
\[
f(x)=x^2,
\qquad x\ge 0.
\]
\end{example}

\begin{solution}
On the domain \([0,\infty)\), the function \(f(x)=x^2\) is one-to-one. Let
\[
y=x^2,
\qquad x\ge 0.
\]
Solving for \(x\), we obtain
\[
x=\sqrt{y}.
\]
We choose the nonnegative square root because the original domain requires \(x\ge 0\). Interchanging \(x\) and \(y\), we obtain
\[
f^{-1}(x)=\sqrt{x},
\qquad x\ge 0.
\]
This example shows that a function that is not one-to-one on all of \(\mathbb{R}\) may still have an inverse after its domain is restricted appropriately.
\end{solution}

\section{Inverse Trigonometric Functions}

The trigonometric functions are not one-to-one on all of \(\mathbb{R}\), so they do not have inverse functions on their full domains. Inverse trigonometric functions arise whenever we know a ratio and want to recover an angle. This occurs in elementary geometry, surveying, navigation, and physics. For example, if we know the slope of a ramp, then \(\arctan\) can be used to determine its angle of inclination.

To define an inverse trigonometric function, we must restrict the domain of the original trigonometric function to an interval on which it is one-to-one. The chosen interval determines the principal values of the inverse function.

Consider the sine function on all of \(\mathbb{R}\). As shown in \cref{fig:sine-not-one-to-one}, a horizontal line can intersect the graph of \(y=\sin x\) more than once. Therefore \(\sin x\) is not one-to-one on \(\mathbb{R}\), and so it does not have an inverse function on its full domain.

To define the inverse sine function, we restrict \(\sin x\) to the interval
\[
-\frac{\pi}{2}\le x\le \frac{\pi}{2}.
\]
On this interval, \(\sin x\) is one-to-one, and every value in the range \([-1,1]\) is attained. The restricted branch is shown in \cref{fig:restricted-sine}.

\begin{definition}
The \emph{inverse sine function}\index{inverse sine function}\index{arcsine@$\arcsin$}, denoted by \(\arcsin x\), is defined by
\[
\arcsin x=y
\quad \Longleftrightarrow \quad
\sin y=x
\quad \text{and} \quad
-\frac{\pi}{2}\le y\le \frac{\pi}{2}.
\]
\end{definition}

The domain of \(\arcsin x\) is \([-1,1]\), and its range is \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\).

\begin{remark}
The notation \(\sin^{-1}x\) is also used for \(\arcsin x\), but it does not mean \((\sin x)^{-1}=1/\sin x\).
\end{remark}

\begin{proposition}
For \(-1\le x\le 1\),
\[
\sin(\arcsin x)=x.
\]
For \(-\frac{\pi}{2}\le x\le \frac{\pi}{2}\),
\[
\arcsin(\sin x)=x.
\]
\end{proposition}

\begin{remark}
The identity \(\arcsin(\sin x)=x\) holds only when \(x\) lies in the principal interval \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\). For example,
\[
\arcsin(\sin \pi)=\arcsin(0)=0\ne \pi.
\]
This is because \(\arcsin x\) always returns a value in \(\left[-\frac{\pi}{2},\frac{\pi}{2}\right]\).
\end{remark}

\begin{example}
Evaluate the following expressions.
\begin{enumerate}
    \item \(\arcsin\left(\frac{1}{2}\right)\)
    \item \(\tan\left(\arcsin\left(\frac{1}{3}\right)\right)\)
\end{enumerate}
\end{example}

\begin{solution}
\begin{enumerate}
    \item We seek the number \(y\) in the interval
    \[
    -\frac{\pi}{2}\le y\le \frac{\pi}{2}
    \]
    such that
    \[
    \sin y=\frac{1}{2}.
    \]
    Since
    \[
    \sin\left(\frac{\pi}{6}\right)=\frac{1}{2},
    \]
    it follows that
    \[
    \arcsin\left(\frac{1}{2}\right)=\frac{\pi}{6}.
    \]

    \item Let
    \[
    \theta=\arcsin\left(\frac{1}{3}\right).
    \]
    Then
    \[
    \sin\theta=\frac{1}{3},
    \qquad
    -\frac{\pi}{2}\le \theta\le \frac{\pi}{2}.
    \]
    Since \(\sin\theta>0\), we take \(\theta\) in the first quadrant. Draw the right triangle with hypotenuse \(3\) and opposite side \(1\), and let \(a\) denote the adjacent side.

    \begin{center}
    \begin{tikzpicture}[scale=1.1]
    \coordinate (A) at (0,0);
    \coordinate (B) at (3,0);
    \coordinate (C) at (3,1);
    \draw[thick] (A) -- (B) -- (C) -- cycle;
    \draw (2.75,0) -- (2.75,0.25) -- (3,0.25);
    \node[left] at (1.2,0.18) {\(\theta\)};
    \node[below] at (1.5,0) {\(a\)};
    \node[right] at (3,0.5) {\(1\)};
    \node[above left] at (1.45,0.5) {\(3\)};
    \end{tikzpicture}
    \end{center}

    By the Pythagorean Theorem,
    \[
    a=\sqrt{3^{2}-1^{2}}=\sqrt{8}=2\sqrt{2}.
    \]
    Therefore,
    \[
    \tan\theta=\frac{1}{a}=\frac{1}{2\sqrt{2}}.
    \]
    Hence
    \[
    \tan\left(\arcsin\left(\frac{1}{3}\right)\right)=\frac{1}{2\sqrt{2}}.
    \]
\end{enumerate}
\end{solution}

\cos(\arccos x)=x.
\]
For \(0\le x\le \pi\),
\[
\arccos(\cos x)=x.
\]
\end{proposition}

\begin{definition}
The \emph{inverse tangent function}\index{inverse tangent function}\index{arctangent@$\arctan$}, denoted by \(\arctan x\), is defined by
\[
\arctan x=y
\quad \Longleftrightarrow \quad
\tan y=x
\quad \text{and} \quad
-\frac{\pi}{2}<y<\frac{\pi}{2}.
\]
\end{definition}

The domain of \(\arctan x\) is \(\mathbb{R}\), and its range is \(\left(-\frac{\pi}{2},\frac{\pi}{2}\right)\).

\begin{proposition}
For \(x\in\mathbb{R}\),
\[
\tan(\arctan x)=x.
\]
For \(-\frac{\pi}{2}<x<\frac{\pi}{2}\),
\[
\arctan(\tan x)=x.
\]
\end{proposition}

\begin{example}
Simplify \(\cos(\arctan x)\).
\end{example}

\begin{solution}
Let
\[
y=\arctan x.
\]
Then
\[
\tan y=x,
\qquad
-\frac{\pi}{2}<y<\frac{\pi}{2}.
\]
We want to find \(\cos y\). Using the identity
\[
1+\tan^{2}y=\sec^{2}y,
\]
we obtain \(\sec^{2}y=1+x^{2}\). Since \(-\frac{\pi}{2}<y<\frac{\pi}{2}\), we have \(\cos y>0\) and hence \(\sec y>0\). Therefore,
\[
\sec y=\sqrt{1+x^{2}}.
\]
It follows that
\[
\cos y=\frac{1}{\sec y}=\frac{1}{\sqrt{1+x^{2}}}.
\]
Thus
\[
\cos(\arctan x)=\frac{1}{\sqrt{1+x^{2}}}.
\]
\end{solution}

\subsection{The remaining inverse trigonometric functions}

The remaining inverse trigonometric functions are used less frequently, but they are defined in the same spirit.

\begin{definition}
The \emph{inverse cosecant function}\index{inverse cosecant function}\index{arccosecant@$\arccsc$}, denoted by \(\arccsc x\), is defined by
\[
\arccsc x=y
\quad \Longleftrightarrow \quad
\csc y=x
\quad \text{and} \quad
y\in \left(0,\frac{\pi}{2}\right]\cup\left(\pi,\frac{3\pi}{2}\right].
\]
\end{definition}

\begin{definition}
The \emph{inverse secant function}\index{inverse secant function}\index{arcsecant@$\arcsec$}, denoted by \(\arcsec x\), is defined by
\[
\arcsec x=y
\quad \Longleftrightarrow \quad
\sec y=x
\quad \text{and} \quad
y\in \left[0,\frac{\pi}{2}\right)\cup\left[\pi,\frac{3\pi}{2}\right).
\]
\end{definition}

\begin{definition}
The \emph{inverse cotangent function}\index{inverse cotangent function}\index{arccotangent@$\arccot$}, denoted by \(\arccot x\), is defined by
\[
\arccot x=y
\quad \Longleftrightarrow \quad
\cot y=x
\quad \text{and} \quad
y\in (0,\pi).
\]
\end{definition}

The domains of \(\arccsc x\) and \(\arcsec x\) are \((-\infty,-1]\cup[1,\infty)\), while the domain of \(\arccot x\) is \(\mathbb{R}\). The principal ranges are the intervals specified in the definitions above.

\begin{remark}
For \(\arcsec x\), we can also choose \(y\in \left[0,\frac{\pi}{2}\right)\cup\left(\frac{\pi}{2},\pi\right]\). In this text we use \(y\in \left[0,\frac{\pi}{2}\right)\cup\left[\pi,\frac{3\pi}{2}\right)\) because later chapters will use derivatives of inverse trigonometric functions, and with this branch choice the differentiation formulas are simpler. When consulting other sources or software, always check which convention is being used, since different branch choices can lead to different values.
\end{remark}

\begin{remark}
For \(\arccsc x\), another common choice is \(y\in \left[-\frac{\pi}{2},0\right)\cup\left(0,\frac{\pi}{2}\right]\). In this text we use \(y\in \left(0,\frac{\pi}{2}\right]\cup\left(\pi,\frac{3\pi}{2}\right]\) so that the chosen branch parallels the branch selected for \(\arcsec x\). Once the principal range is fixed consistently, either convention gives a valid inverse function.
\end{remark}

\section{The Limit of a Function}

In many situations, we are interested not only in the value of a function at a point, but also in the behavior of the function when the input is close to that point. One important example comes from the study of motion.

Suppose that \(s(t)\) denotes the position of an object at time \(t\). Then the average velocity of the object from time \(t\) to time \(t+h\) is
\[
\frac{s(t+h)-s(t)}{h}.
\]
More generally, for a function \(f\), the ratio
\[
\frac{f(y)-f(x)}{y-x}
\qquad \text{or} \qquad
\frac{f(x+h)-f(x)}{h}
\]
measures the average rate of change of the function over an interval.

However, in many applications we do not want the average rate of change over an interval; we want the rate of change at a single instant. For example, in mechanics we distinguish between average velocity and instantaneous velocity. The average velocity describes the motion over a time interval, whereas the instantaneous velocity describes how fast the object is moving at one particular moment. To understand this instantaneous behavior, we let the second point move closer and closer to the first point. In other words, we study what happens when the input is very close to the point of interest. This leads naturally to the concept of the limit of a function.

\begin{definition}\label{def:limit-informal}
Suppose that \(f(x)\) is defined for all \(x\) near \(a\), except possibly at \(x=a\). We write
\[
\lim_{x\to a} f(x)=L
\]
and say that the \emph{limit}\index{limit}\index{limit@$\lim$} of \(f(x)\) as \(x\) approaches \(a\) is \(L\) if \(f(x)\) can be made arbitrarily close to \(L\) whenever \(x\) is sufficiently close to \(a\) but not equal to \(a\) \((x\ne a)\).
\end{definition}

This is an informal description; the phrases ``arbitrarily close'' and ``sufficiently close'' are intuitive but not yet rigorous. We will replace them with the precise \(\varepsilon\)-\(\delta\) formulation in \cref{def:limit-precise}.

When we consider \(\lim_{x\to a} f(x)\), we are concerned with the values of \(f(x)\) for \(x\) near \(a\), not necessarily with the value of \(f(a)\). In particular, \(f(a)\) need not be defined at all.

\bottomrule

\[
\frac{x-1}{x^{2}-1}
\]
gets close to \(\frac{1}{2}\) as \(x\) approaches \(1\). Thus we guess that
\[
\lim_{x\to 1}\frac{x-1}{x^{2}-1}=\frac{1}{2}.
\]
\end{solution}

\begin{example}

\[
\lim_{t\to 0}\frac{\sqrt{t^{2}+9}-3}{t^{2}}.
\]
\end{example}

\begin{solution}

\[
\begin{array}{c cccc}
\toprule

\midrule
\tfrac{\sqrt{t^{2}+9}-3}{t^{2}} & 0.166620 & 0.166666 & 0.166666 & 0.166620 \\
\bottomrule
\end{array}
\]
These values are close to
\[
\frac{1}{6}=0.166666\cdots
\]
so we guess that
\[
\lim_{t\to 0}\frac{\sqrt{t^{2}+9}-3}{t^{2}}=\frac{1}{6}.
\]
\end{solution}

The examples above show that the value of a limit can often be guessed by looking at function values near the point of interest. In the following sections, we will develop systematic methods for evaluating limits and study more subtle situations, such as one-sided limits and infinite limits.

\section{One-Sided Limits, Infinite Limits, and Vertical Asymptotes}

In the previous section, we introduced the idea of the limit of a function. Sometimes, however, the behavior of a function from the left and from the right is different. This leads to the notion of one-sided limits. We also study situations in which the values of a function become arbitrarily large positive or arbitrarily large negative near a point.

\subsection{One-sided limits}

\begin{definition}
We write
\[
\lim_{x\to a^-}f(x)=L
\]
and say that the \emph{left-hand limit}\index{left-hand limit}\index{limit!left-hand} of \(f(x)\) as \(x\) approaches \(a\) is \(L\) if \(f(x)\) approaches \(L\) as \(x\) approaches \(a\) through values less than \(a\).

Similarly, we write
\[
\lim_{x\to a^+}f(x)=L
\]
and say that the \emph{right-hand limit}\index{right-hand limit}\index{limit!right-hand} of \(f(x)\) as \(x\) approaches \(a\) is \(L\) if \(f(x)\) approaches \(L\) as \(x\) approaches \(a\) through values greater than \(a\).
\end{definition}

\begin{proposition}[Criterion for a Two-Sided Limit]
\[
\lim_{x\to a}f(x)=L
\]
if and only if
\[
\lim_{x\to a^-}f(x)=L
\qquad \text{and} \qquad
\lim_{x\to a^+}f(x)=L.
\]
\end{proposition}

If the left-hand limit and the right-hand limit are different, then the two-sided limit does not exist.

\begin{example}
Let
\[
f(x)=
\begin{cases}
x+1, & x<2,\\[4pt]
4-x, & x\ge 2.
\end{cases}
\]
Evaluate \(\lim_{x\to 2^-}f(x)\), \(\lim_{x\to 2^+}f(x)\), and determine whether \(\lim_{x\to 2}f(x)\) exists.
\end{example}

\begin{solution}
If \(x\to 2^{-}\), then we use the branch \(f(x)=x+1\). Therefore,
\[
\lim_{x\to 2^-}f(x)=\lim_{x\to 2^-}(x+1)=3.
\]

If \(x\to 2^{+}\), then we use the branch \(f(x)=4-x\). Therefore,
\[
\lim_{x\to 2^+}f(x)=\lim_{x\to 2^+}(4-x)=2.
\]

Since the left-hand limit and the right-hand limit are different, the two-sided limit does not exist. Hence
\[
\lim_{x\to 2}f(x)
\]
does not exist.
\end{solution}

\subsection{Infinite limits}

In some cases, the values of \(f(x)\) do not approach a finite number as \(x\to a\). Instead, they become arbitrarily large positive or arbitrarily large negative.

\begin{definition}
Let \(f\) be a function defined on both sides of \(a\), except possibly at \(a\) itself. We write
\[
\lim_{x\to a}f(x)=\infty
\]
if the values of \(f(x)\) can be made arbitrarily large by taking \(x\) sufficiently close to \(a\), with \(x\ne a\). In this case we call \(\lim_{x\to a}f(x)\) an \emph{infinite limit}\index{infinite limit}\index{limit!infinite}.

Similarly, we write
\[
\lim_{x\to a}f(x)=-\infty
\]
if the values of \(f(x)\) can be made arbitrarily large negative by taking \(x\) sufficiently close to \(a\), with \(x\ne a\).
\end{definition}

The symbols \(\infty\) and \(-\infty\) do not represent real numbers. They indicate that a function grows without bound (positively or negatively) near a specific point.

Similar definitions apply to one-sided infinite limits:
\[
\lim_{x\to a^-}f(x)=\infty,
\qquad
\lim_{x\to a^+}f(x)=\infty,
\qquad
\lim_{x\to a^-}f(x)=-\infty,
\qquad
\lim_{x\to a^+}f(x)=-\infty.
\]

\begin{example}
Consider the function
\(
f(x)=\dfrac{1}{x^2}.
\)
Guess the value of
\[
\lim_{x\to 0}\frac{1}{x^2}.
\]
\end{example}

\begin{solution}
If \(x\) is close to \(0\), then \(x^2\) is a very small positive number. Therefore \(\dfrac{1}{x^2}\) becomes very large. For example,
\[
\frac{1}{(0.1)^2}=100,
\qquad
\frac{1}{(0.01)^2}=10000.
\]
Thus the values of \(\dfrac{1}{x^2}\) increase without bound as \(x\to 0\). Hence
\[
\lim_{x\to 0}\frac{1}{x^2}=\infty.
\]
\end{solution}

\begin{example}
Guess the value of
\[
\lim_{x\to 0}\left(-\frac{1}{x^2}\right).
\]
\end{example}

\begin{solution}
Since \(\dfrac{1}{x^2}\) becomes arbitrarily large positive as \(x\to 0\), the quantity \(-\dfrac{1}{x^2}\) becomes arbitrarily large negative. Therefore
\[
\lim_{x\to 0}\left(-\frac{1}{x^2}\right)=-\infty.
\]
\end{solution}

\subsection{Vertical asymptotes}

Infinite limits are closely related to vertical asymptotes.

\begin{definition}
The vertical line \(x=a\) is called a \emph{vertical asymptote}\index{vertical asymptote}\index{asymptote!vertical} of the curve \(y=f(x)\) if at least one of the following statements is true:
\[
\lim_{x\to a^-}f(x)=\infty,
\qquad
\lim_{x\to a^+}f(x)=\infty,
\qquad
\lim_{x\to a^-}f(x)=-\infty,
\qquad
\lim_{x\to a^+}f(x)=-\infty.
\]
\end{definition}

If \(x=a\) is a vertical asymptote, then the function values become arbitrarily large positive or arbitrarily large negative as \(x\) approaches \(a\) from at least one side.

\begin{example}
Evaluate
\[
\lim_{x\to 0^-}\frac{1}{x},
\qquad
\lim_{x\to 0^+}\frac{1}{x},
\]
determine whether
\[
\lim_{x\to 0}\frac{1}{x}
\]
exists, and decide whether \(x=0\) is a vertical asymptote of the graph of \(y=\dfrac{1}{x}\).
\end{example}

\begin{solution}
If \(x\to 0^{-}\), then \(x\) is negative and very close to \(0\), so \(\dfrac{1}{x}\) is a large negative number. Hence
\[
\lim_{x\to 0^-}\frac{1}{x}=-\infty.
\]

If \(x\to 0^{+}\), then \(x\) is positive and very close to \(0\), so \(\dfrac{1}{x}\) is a large positive number. Hence
\[
\lim_{x\to 0^+}\frac{1}{x}=\infty.
\]

Since the left-hand and right-hand limits are different, the two-sided limit \(\lim_{x\to 0}\dfrac{1}{x}\) does not exist. However, at least one one-sided infinite limit occurs, so the line \(x=0\) is a vertical asymptote of the graph of \(y=\dfrac{1}{x}\).
\end{solution}

\begin{example}
Does the curve \(y=\dfrac{2x}{x-3}\) have a vertical asymptote?
\end{example}

\begin{solution}
The denominator is zero when \(x=3\), so \(x=3\) is the only possible vertical asymptote.

If \(x\) is close to \(3\) and \(x>3\), then \(x-3\) is a small positive number and \(2x\) is close to \(6\). Therefore \(\dfrac{2x}{x-3}\) is a large positive number, and hence
\[
\lim_{x\to 3^+}\frac{2x}{x-3}=\infty.
\]
If \(x\) is close to \(3\) and \(x<3\), then \(x-3\) is a small negative number while \(2x\) is still close to \(6\). Therefore \(\dfrac{2x}{x-3}\) is a large negative number, and hence
\[
\lim_{x\to 3^-}\frac{2x}{x-3}=-\infty.
\]
Therefore the line \(x=3\) is a vertical asymptote of the curve, as shown in \cref{fig:vertical-asymptote-example}.
\end{solution}

\[
\lim_{x\to 0^+}\ln x=-\infty.
\]

\[

\]

\section{Limit Laws and Computational Techniques}

In the previous sections, we used tables and graphs to guess the values of limits. We now introduce algebraic tools for evaluating limits more systematically. The basic idea is that limits behave well with respect to the usual algebraic operations.

\subsection{The limit laws}

\begin{theorem}[Limit laws]\index{limit laws}
Suppose that \(c\) is a constant and that
\[
\lim_{x\to a}f(x)
\qquad \text{and} \qquad
\lim_{x\to a}g(x)
\]
both exist. Then:
\[
\lim_{x\to a}\bigl(f(x)+g(x)\bigr)
=
\lim_{x\to a}f(x)+\lim_{x\to a}g(x),
\]
\[
\lim_{x\to a}\bigl(f(x)-g(x)\bigr)
=
\lim_{x\to a}f(x)-\lim_{x\to a}g(x),
\]
\[
\lim_{x\to a}\bigl(c f(x)\bigr)
=
c\lim_{x\to a}f(x),
\]
\[
\lim_{x\to a}\bigl(f(x)g(x)\bigr)
=
\left(\lim_{x\to a}f(x)\right)\left(\lim_{x\to a}g(x)\right),
\]
and
\[
\lim_{x\to a}\frac{f(x)}{g(x)}
=
\frac{\lim_{x\to a}f(x)}{\lim_{x\to a}g(x)},
\qquad
\text{provided that } \lim_{x\to a}g(x)\ne 0.
\]
\end{theorem}

These laws can be summarized verbally as follows:
\begin{enumerate}
    \item The limit of a sum is the sum of the limits.
    \item The limit of a difference is the difference of the limits.
    \item The limit of a constant multiple is the constant times the limit.
    \item The limit of a product is the product of the limits.
    \item The limit of a quotient is the quotient of the limits, provided the denominator limit is not \(0\).
\end{enumerate}

\begin{remark}
These properties will be proved rigorously later. For now, they provide an effective way to evaluate many limits.
\end{remark}

\subsection{Direct substitution}

The limit laws imply several useful consequences.

\begin{proposition}
Let \(n\) be a positive integer. Then
\[
\lim_{x\to a}x=a
\qquad \text{and} \qquad
\lim_{x\to a}x^{n}=a^{n}.
\]
Consequently, if \(p(x)\) is a polynomial, then
\[
\lim_{x\to a}p(x)=p(a).
\]
If \(r(x)\) is a rational function and \(r(a)\) is defined, then
\[
\lim_{x\to a}r(x)=r(a).
\]
\end{proposition}

So for polynomials and rational functions, the first step is usually direct substitution. If substitution gives a meaningful number, then the limit is found immediately.

\subsection{Algebraic simplification}

Sometimes direct substitution does not work because it leads to an expression such as \(\frac{0}{0}\). In that case, we try to simplify the expression first.

\begin{example}
Evaluate
\[
\lim_{x\to 1}\frac{x-1}{x^{2}-1}.
\]
\end{example}

\begin{solution}
Direct substitution gives
\[
\frac{1-1}{1^{2}-1}=\frac{0}{0},
\]
so we first simplify the expression. Since
\[
x^{2}-1=(x-1)(x+1),
\]
we have
\[
\frac{x-1}{x^{2}-1}
=
\frac{x-1}{(x-1)(x+1)}
=
\frac{1}{x+1},
\qquad x\ne 1.
\]
Therefore,
\[
\lim_{x\to 1}\frac{x-1}{x^{2}-1}
=
\lim_{x\to 1}\frac{1}{x+1}
=
\frac{1}{2}.
\]
\end{solution}

\begin{example}
Evaluate
\[
\lim_{t\to 0}\frac{\sqrt{t^{2}+9}-3}{t^{2}}.
\]
\end{example}

\begin{solution}
Direct substitution again gives the indeterminate form \(\frac{0}{0}\). We rationalize the numerator:
\[
\frac{\sqrt{t^{2}+9}-3}{t^{2}}
\cdot
\frac{\sqrt{t^{2}+9}+3}{\sqrt{t^{2}+9}+3}
=
\frac{t^{2}+9-9}{t^{2}\bigl(\sqrt{t^{2}+9}+3\bigr)}.
\]
Thus
\[
\frac{\sqrt{t^{2}+9}-3}{t^{2}}
=
\frac{1}{\sqrt{t^{2}+9}+3},
\qquad t\ne 0.
\]
Now we may take the limit directly:
\[
\lim_{t\to 0}\frac{\sqrt{t^{2}+9}-3}{t^{2}}
=
\lim_{t\to 0}\frac{1}{\sqrt{t^{2}+9}+3}
=
\frac{1}{6}.
\]
\end{solution}

\subsection{Limits of piecewise-defined functions}

Not all limits can be evaluated by direct substitution alone. Sometimes we must use the definition of the function more carefully.

\begin{example}
Evaluate
\[
\lim_{x\to 0}|x|.
\]
\end{example}

\begin{solution}
Recall that
\[
|x|=
\begin{cases}
x, & x\ge 0,\\[4pt]
-x, & x<0.
\end{cases}
\]
If \(x\to 0^{+}\), then \(|x|=x\to 0\). If \(x\to 0^{-}\), then \(|x|=-x\to 0\). Therefore,
\[
\lim_{x\to 0}|x|=0.
\]
\end{solution}

\begin{example}
The greatest integer function, also called the floor function, is denoted by \(\lfloor x \rfloor\) and defined to be the largest integer less than or equal to \(x\). Evaluate
\[
\lim_{x\to 3^{+}}\lfloor x \rfloor
\qquad \text{and} \qquad
\lim_{x\to 3^{-}}\lfloor x \rfloor.
\]
\end{example}

\begin{solution}
If \(x\to 3^{+}\), then \(x\) is slightly greater than \(3\), so
\[
\lfloor x \rfloor=3.
\]
Hence
\[
\lim_{x\to 3^{+}}\lfloor x \rfloor=3.
\]

If \(x\to 3^{-}\), then \(x\) is slightly less than \(3\), so
\[
\lfloor x \rfloor=2.
\]
Hence
\[
\lim_{x\to 3^{-}}\lfloor x \rfloor=2.
\]

Since the left-hand and right-hand limits are different, we conclude that
\(
\lim_{x\to 3}\lfloor x \rfloor
\)
does not exist.
\end{solution}

\subsection{The squeeze theorem}

A very useful technique for evaluating limits is to compare a function with two simpler functions.

\begin{theorem}[Squeeze theorem]\index{squeeze theorem}
Suppose that
\[
g(x)\le f(x)\le h(x)
\]
for all \(x\) near \(a\), except possibly at \(x=a\). If
\[
\lim_{x\to a}g(x)=L
\qquad \text{and} \qquad
\lim_{x\to a}h(x)=L,
\]
then
\[
\lim_{x\to a}f(x)=L.
\]
\end{theorem}

\begin{example}
Evaluate
\[
\lim_{x\to 0}x^{2}\sin\left(\frac{1}{x}\right).
\]
\end{example}

\begin{solution}
Since
\[
-1\le \sin\left(\frac{1}{x}\right)\le 1,
\]
for all \(x\ne 0\), we may multiply all parts by \(x^{2}\ge 0\) to obtain
\[
-x^{2}\le x^{2}\sin\left(\frac{1}{x}\right)\le x^{2}.
\]
This inequality holds for all \(x\ne 0\). Now,
\[
\lim_{x\to 0}(-x^{2})=0
\qquad \text{and} \qquad
\lim_{x\to 0}x^{2}=0.
\]
Therefore, by the squeeze theorem,
\[
\lim_{x\to 0}x^{2}\sin\left(\frac{1}{x}\right)=0.
\]
\end{solution}

\begin{example}
Evaluate
\[
\lim_{x\to 0}x\cos\left(\frac{1}{x}\right).
\]
\end{example}

\begin{solution}
Since
\[
\left|\cos\left(\frac{1}{x}\right)\right|\le 1,
\]
for all \(x\ne 0\), we have
\[
\left|x\cos\left(\frac{1}{x}\right)\right|
\le |x|.
\]
Equivalently,
\[
-|x|\le x\cos\left(\frac{1}{x}\right)\le |x|.
\]
This inequality also holds for all \(x\ne 0\). Now
\[
\lim_{x\to 0}(-|x|)=0
\qquad \text{and} \qquad
\lim_{x\to 0}|x|=0.
\]
Therefore, by the squeeze theorem,
\[
\lim_{x\to 0}x\cos\left(\frac{1}{x}\right)=0.
\]
\end{solution}

The limit laws and the techniques of factoring, rationalization, and squeezing allow us to evaluate many limits. In the next section, we will give the precise definition of a limit.

\section{The Precise Definition of a Limit}

From the time of Newton and Leibniz, calculus was remarkably successful in solving problems in geometry, mechanics, and astronomy. However, for a long time, there was also controversy about its logical foundations. Early arguments often relied on infinitesimals---quantities thought of as being extremely small, but not described in a fully precise way. As a result, mathematicians continued to ask what it really means to say that \(x\) approaches \(a\) or that \(f(x)\) approaches \(L\).

In the previous sections, we used graphs, tables, and algebraic techniques to study limits. However, informal expressions such as ``\(x\) is close to \(a\)'' and ``\(f(x)\) is close to \(L\)'' are not rigorous enough. The modern resolution to this difficulty is the \(\varepsilon\)-\(\delta\) definition\index{epsilon-delta definition@$\varepsilon$-$\delta$ definition} of a limit, developed in the nineteenth century, which gave calculus the level of rigor used in modern mathematics. Instead of relying on vague language, it describes exactly what it means for \(f(x)\) to be close to \(L\) whenever \(x\) is sufficiently close to \(a\). We now introduce this precise definition.

To motivate the definition, consider the function
\[
f(x)=
\begin{cases}
2x-1, & x\ne 3,\\[4pt]
6, & x=3.
\end{cases}
\]
When \(x\) is close to \(3\), but \(x\ne 3\), the value of \(f(x)=2x-1\) is close to \(5\). Thus it is natural to expect that
\(
\lim_{x\to 3} f(x)=5.
\)

\begin{example}
For the function
\[
f(x)=
\begin{cases}
2x-1, & x\ne 3,\\[4pt]
6, & x=3,
\end{cases}
\]
how close must \(x\) be to \(3\) in order to guarantee that \(f(x)\) differs from \(5\) by less than \(0.1\)?
\end{example}

\begin{solution}
We want
\[
|f(x)-5|<0.1.
\]
Since \(f(x)=2x-1\) for \(x\ne 3\), we have
\[
|f(x)-5|
=
|(2x-1)-5|
=
|2x-6|
=
2|x-3|.
\]
Therefore, to ensure that
\[
|f(x)-5|<0.1,
\]
it is enough to require that
\[
2|x-3|<0.1.
\]
This gives
\[
|x-3|<0.05.
\]
So if \(x\) is within \(0.05\) of \(3\), but \(x\ne 3\), then \(f(x)\) is within \(0.1\) of \(5\).
\end{solution}

If we replace \(0.1\) by the smaller number \(0.01\), then the same reasoning shows that
\[
|f(x)-5|<0.01
\qquad \text{whenever} \qquad
0<|x-3|<0.005.
\]
Thus, when we ask for \(f(x)\) to be closer to \(5\), we must require \(x\) to be closer to \(3\).

More generally, if we want
\[
|f(x)-5|<\varepsilon
\]
for some \(\varepsilon>0\), then it is enough to require that
\[
2|x-3|<\varepsilon,
\]
that is,
\[
|x-3|<\frac{\varepsilon}{2}.
\]
Thus for this example we may choose
\[
\delta=\frac{\varepsilon}{2}.
\]

The previous discussion shows the basic idea. We want to control the distance between \(f(x)\) and the limit \(L\) by controlling the distance between \(x\) and the point \(a\).

\begin{definition}\label{def:limit-precise}
Let \(f\) be a function defined on an open interval containing \(a\), except possibly at \(a\) itself. We say that
\(
\lim_{x\to a} f(x)=L
\)
if for every \(\varepsilon>0\) there exists a number \(\delta>0\) such that
\[
|f(x)-L|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-a|<\delta.
\]
\end{definition}

This is the rigorous counterpart of the informal \cref{def:limit-informal}: ``\(f(x)\) is arbitrarily close to \(L\)'' becomes ``\(|f(x)-L|<\varepsilon\) for every \(\varepsilon>0\),'' and ``\(x\) is sufficiently close to \(a\), but \(x\ne a\)'' becomes ``\(0<|x-a|<\delta\) for a suitably chosen \(\delta\).''

\Cref{fig:precise-limit} illustrates the definition. If \(x\) is restricted to lie in the interval
\((a-\delta,a+\delta)\), \(x\ne a\),
then the corresponding values of \(f(x)\) are forced to lie in the interval
\((L-\varepsilon,L+\varepsilon)\).
Notice that the value \(f(a)\) itself may be different from \(L\). The limit depends only on the behavior of \(f(x)\) when \(x\) is close to \(a\), not necessarily on the value of the function at \(x=a\).

The condition \( 0<|x-a|<\delta \) means that \(x\) is within a distance \(\delta\) of \(a\), but \(x\ne a\). The inequality \( |f(x)-L|<\varepsilon \) means that \(f(x)\) is within a distance \(\varepsilon\) of \(L\). Moreover, the statement \( \lim_{x\to a} f(x)=L \) must hold for every \(\varepsilon>0\), no matter how small \(\varepsilon\) is.

\begin{proposition}[Uniqueness of Limits]
If \(\lim_{x\to a} f(x)\) exists, then it is unique.
\end{proposition}

\begin{proof}
Suppose that
\[
\lim_{x\to a} f(x)=L
\qquad \text{and} \qquad
\lim_{x\to a} f(x)=M.
\]
If \(L\ne M\), let
\[
\varepsilon=\frac{|L-M|}{2}.
\]
Then there exist \(\delta_1,\delta_2>0\) such that
\[
|f(x)-L|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-a|<\delta_1,
\]
and
\[
|f(x)-M|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-a|<\delta_2.
\]
If \(0<|x-a|<\min\{\delta_1,\delta_2\}\), then both inequalities hold. By the triangle inequality,
\[
|L-M|
=
|L-f(x)+f(x)-M|
\le
|L-f(x)|+|f(x)-M|
<
2\varepsilon
=
|L-M|,
\]
which is impossible. Therefore \(L=M\).
\end{proof}

\begin{example}
Show from the definition that
\[
\lim_{x\to 3} (4x-5)=7.
\]
\end{example}

\begin{solution}
Given \(\varepsilon>0\), we want to find \(\delta>0\) such that
\[
|(4x-5)-7|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-3|<\delta.
\]
Now
\[
|(4x-5)-7|
=
|4x-12|
=
4|x-3|.
\]
So if we require \(4|x-3|<\varepsilon\), it is enough to have \(|x-3|<\dfrac{\varepsilon}{4}\). Hence we may choose
\[
\delta=\frac{\varepsilon}{4}.
\]
Then, whenever \(0<|x-3|<\delta\), we have
\[
|(4x-5)-7|
=
4|x-3|
<
4\left(\frac{\varepsilon}{4}\right)
=
\varepsilon.
\]
Therefore,
\[
\lim_{x\to 3} (4x-5)=7.
\]
\end{solution}

\begin{example}
Show from the definition that
\[
\lim_{x\to 1} (x^{2}-5x+6)=2.
\]
\end{example}

\begin{solution}
Given \(\varepsilon>0\), we want to find \(\delta>0\) such that
\[
|(x^{2}-5x+6)-2|<\varepsilon
\qquad \text{whenever} \qquad
0<|x-1|<\delta.
\]
First we simplify:
\[
|(x^{2}-5x+6)-2|
=
|x^{2}-5x+4|
=
|(x-1)(x-4)|.
\]
The difficulty is that the factor \(|x-4|\) also depends on \(x\). To control it, we first assume \(|x-1|<1\). Then
\begin{align*}
|x-1|<1
&\implies -1<x-1<1 \\
&\implies 0<x<2 \\
&\implies -4<x-4<-2 \\
&\implies |x-4|<4.
\end{align*}
Thus, whenever \(|x-1|<1\), we have
\[
|(x-1)(x-4)|<4|x-1|.
\]
So to guarantee that \(|(x-1)(x-4)|<\varepsilon\), it is enough to require \(4|x-1|<\varepsilon\), that is, \(|x-1|<\dfrac{\varepsilon}{4}\). Therefore we choose
\[
\delta=\min\left\{1,\frac{\varepsilon}{4}\right\}.
\]
Then, whenever \(0<|x-1|<\delta\), we have both \(|x-1|<1\) and \(|x-1|<\dfrac{\varepsilon}{4}\), so \(|x-4|<4\) and
\[
|(x^{2}-5x+6)-2|
=
|(x-1)(x-4)|
<
4|x-1|
<
4\left(\frac{\varepsilon}{4}\right)
=
\varepsilon.
\]
Therefore,
\[
\lim_{x\to 1} (x^{2}-5x+6)=2.
\]
\end{solution}

\begin{definition}
We write
\(
\lim_{x\to a} f(x)=\infty
\)
if for every positive number \(M\), there exists \(\delta>0\) such that
\[
f(x)>M
\qquad \text{whenever} \qquad
0<|x-a|<\delta.
\]
\end{definition}

\begin{definition}
We write
\(
\lim_{x\to a} f(x)=-\infty
\)
if for every negative number \(N\), there exists \(\delta>0\) such that
\[
f(x)<N
\qquad \text{whenever} \qquad
0<|x-a|<\delta.
\]
\end{definition}

In the first definition we may think of \(M\) as any large positive threshold, such as \(100\), \(1000\), \(10000\), and so on: no matter how large a bound \(M\) we fix, \(f(x)\) eventually exceeds it whenever \(x\) is close enough to \(a\). Similarly, in the second definition we think of \(N\) as any negative threshold, such as \(-100\), \(-1000\), \(-10000\), and so on, below which \(f(x)\) is eventually forced.

The \(\varepsilon\)-\(\delta\) formulation also allows us to give a precise meaning to a property of functions that will appear throughout the chapters to come. If \(f\) is defined at every point near \(a\), including \(a\), and if
\[
\lim_{x\to a} f(x)=f(a),
\]
then \(f\) is said to be \emph{continuous at \(a\)}\index{continuous function}\index{continuity}. We mention this term here only because it is expressed directly in terms of limits; a systematic study of continuity, together with its main properties and theorems, belongs to a later chapter.
